## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/AhmedHassan222/Sakan.git
   cd Sakan
2. **Install dependencies:**
   ```bash
   npm install
3. **Start the development server:**
   ```bash
   npm start
